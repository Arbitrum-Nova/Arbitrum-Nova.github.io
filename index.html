What is Arbitrum Nova?

Arbitrum Nova is an Arbitrum Nitro AnyTrust chain with Ethereum as its underlying chain, according to the official Arbitrum chain information. It is EVM-compatible and uses ETH for gas.

How Arbitrum Nova works

  1. Open the bridgeLaunch the official Arbitrum bridge with Ethereum as the source and the destination chain preselected.
  2. Connect a walletUse a Web3 wallet with ETH on the source network for the transfer and gas.
  3. Choose asset and amountSelect ETH or a supported ERC-20, review the route, and keep enough ETH for transaction costs.
  4. Confirm and verifySign in your wallet, then check the destination wallet or explorer after the bridge transaction settles.

Before you bridge

AnyTrust design

Nova uses Arbitrum Nitro AnyTrust, which relies on a Data Availability Committee to reduce data costs versus posting all transaction data to Ethereum; see the AnyTrust protocol docs.

Known network details

Use Chain ID 42170, ETH as the currency symbol, and the public RPC endpoint listed in Arbitrum's chain information when adding the network manually.

Official routing first

Start from the Arbitrum bridge and verify the domain before connecting a wallet or approving a token.

Arbitrum Nova FAQ

What is arbitrum nova?

arbitrum nova is Arbitrum's Nitro AnyTrust Layer 2 with Ethereum as the underlying chain, Chain ID 42170, and ETH as the gas token.

How do I bridge to arbitrum nova?

Use the official Arbitrum bridge, connect your wallet, set Ethereum as the source, choose Arbitrum Nova as the destination, select an asset, and confirm the transaction in your wallet.

What wallet details does arbitrum nova use?

Use network name Arbitrum Nova, RPC URL https://nova.arbitrum.io/rpc, Chain ID 42170, currency symbol ETH, and Blockscout as the current explorer reference.

Is arbitrum nova the same as Arbitrum One?

No. Both are Arbitrum chains, but Nova uses Nitro AnyTrust while Arbitrum One is listed as a Nitro Rollup in the official chain information.

Where can I check arbitrum nova transactions?

Use Arbitrum Nova Blockscout to inspect addresses, blocks, and transactions on the network.
